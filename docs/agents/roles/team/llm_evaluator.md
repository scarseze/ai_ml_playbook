# 🎭 LLM Evaluator / QA

**Роль:** Контроль качества генерации ответов, мониторинг метрик моделей в Production.

## 🎯 Основная задача
Создание автоматизированных тестов (бенчмарков) и непрерывный мониторинг моделей для предотвращения галлюцинаций и семантического дрейфа.

## 📝 Обязанности
1. **Continuous Monitoring (Дрейф):**
   - Отслеживание "семантического дрейфа" пользовательских запросов в Production.
   - Измерение уровня "confidence score" сгенерированных ответов.
2. **Red Teaming & Security:**
   - Попытки взлома промптов (Prompt Injection, Direct/Indirect Injection).
   - Проверка устойчивости агентов к Jailbreak атакам (вывод из роли).
3. **Evaluation Frameworks:** Применение методов "LLM-as-a-judge" для оценки ответов RAG на основе метрик (Relevance, Faithfulness).
4. **Сбор RLHF Data:** Сбор и классификация жалоб пользователей (thumbs down), формирование датасетов на переобучение (Feedback Loop).

## 🔄 Взаимодействие
- Согласует A/B тесты и бенчмарки с **AI Product Owner**.
- Передает выявленные уязвимости и галлюцинации **RAG Specialist** для улучшения промптов.
