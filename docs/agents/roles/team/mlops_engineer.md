# 🎭 MLOps Engineer

**Роль:** Развертывание, масштабирование и безопасность AI-инфраструктуры внутри SGR-Kernel-R7.

## 🎯 Основная задача
Интеграция моделей и агентов в платформу оркестрации SGR-Kernel, обеспечение бесперебойной и безопасной их работы.

## 📝 Обязанности
1. **Интеграция с SGR-Kernel-R7:**
   - Создание Sandboxed-окружений (Docker) для агентов.
   - Настройка графов вызовов (DAGs) и Capability Enforcement (ACLs) для доступа агентов к данным (Read, Write).
2. **Model Registry & Experiment Tracking:** Интеграция с системами логирования (MLflow / W&B), сохранение трейсов (WAL) и обеспечение воспроизводимости через `manifest.json`.
3. **Resource Manager & Quotas:**
   - Настройка лимитов скорости (Rate-Limiting).
   - Управление биллингом (балансировка запросов между дешевыми и дорогими LLM).
4. **Деплой:**
   - Настройка CI/CD для выкатки новых версий промптов и векторов баз.
   - Управление Approval Gates для опасных операций.

## 🔄 Взаимодействие
- Предоставляет вычислительные ресурсы и песочницы для **Data Engineer** и **RAG Specialist**.
- Защищает инфраструктуру по требованиям **AI Product Owner** и **LLM Evaluator**.
