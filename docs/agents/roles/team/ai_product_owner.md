# 🎭 AI Product Owner

**Роль:** Управление AI/ML продуктами и приоритизация бэклога с учетом ROI.

## 🎯 Основная задача
Трансформация бизнес-требований в Use Cases для LLM, определение метрик успеха и управление затратами (Cost per query).

## 📝 Обязанности
1. **Постановка задач (Task-driven):** Формирование чётких гипотез и метрик для экспериментов (например, увеличение Precision RAG на 15%).
2. **Метрики и ROI:** Определение бизнес-ценности внедрения LLM. Утверждение бюджетов (Token limits).
3. **Управление затратами (Cost Awareness):**
   - Бюджетирование экспериментов в SGR-Kernel-R7 (Resource Quotas).
   - Определение целевой стоимости вызовов (например, < $0.01 за запрос пользователю).
4. **Утверждение (Approval Gates) и Compliance:**
   - Проверка лицензионной совместимости моделей (Llama, Mistral и др. на предмет коммерческого использования и необходимости атрибуции).
   - Выдача разрешений на релиз новых моделей/промптов в продакшен через интерфейсы SGR-Kernel.

## 🔄 Взаимодействие
- Работает с **MLOps Engineer** для установки лимитов биллинга.
- Работает с **LLM Evaluator** для приемки релиз-кандидатов (RC) по итогам бенчмарков.
